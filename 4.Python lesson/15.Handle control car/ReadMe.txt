1. Download the program to the micro:bit mainboard of the car. You need to use the MU software to open the microbit-car-code.py file in the course directory, and then click Flash download.

2. Download the program to the microbit mainboard of the remote control handle. You need to use the MU software to open the ghandle-code.py file in the course directory, and then click Flash download.

3. Whether you download the robot or handle program, you need to import the buildingbit library file into the microbit first. If you dont know how to import it, please check the course on importing the buildingbit library prepared before the class.

4. After the program is downloaded, power on the handle and the car normally, they will automatically pair and can be controlled within the signal transmission range.
